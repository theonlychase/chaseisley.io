T3 Sub
by Chase Isley
Project Description
This ecommerce website was built using Wordpress for Subway franchisees. The shop has close to 1500 customizable products and is built with WooCommerce. It utilizes many custom plugins, such as Avalara Tax, UPS shipping, USPS Shipping, and the Authorize.net Gateway. The website is 100% mobile responsive.
