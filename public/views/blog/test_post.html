Reverse a String


This one is fairly simple. There are many ways to solve this, but I am going to show you two different ways that I solved it.

Solution 1

The first way I solved it was using the split(), reverse(), and join() methods. Here is the problem below

Step 1: We want to turn the passed in string into an array. Since we are passing in the string "hello", then we will start out by using the "Split Method".
Since we have a function, we want to return something, so the code would start off with a return statement, which is:

return str.split("");

What this does is splits each letter in the string, and makes an array. This is what is returned:

["h","e","l","l","o"].

Step 2: We want to reverse the elements in the array. We can easily do this with the "Reverse Method".
The code looks like this:

return str.split("").reverse();

This is what is returned:

["o","l","l","e","h"]

Step 3: Now that the array is reversed, our last step is to make the array a String. We do this with the "Join Method".

What the join method does is join the elements of the array into a String. Since our array has been reversed, we simply add join("") to the end of our return statement. What goes inside the parentheses of join("") is called the seperator. By default, the separator is a comma (,). Since we have (""), that means that the array will become a String with no spaces between each element.
The code looks like this:

return str.split("").reverse().join("");

Solution 2

The second way I solved it was using a reverse for loop with concatenation

Step 1: We want to create a new variable and set it to an empty string.

var reverseStr = "";

Step 2: We want to create a for loop, but instead of starting the loop from the beginning of the passed-in string, we want to start from the end of the passed-in string. We set it up like this:

for (var i = str.length - 1; i >= 0; i--)

Remember, a string has a 0 based index, which is why we set variable i to str.length - 1. str.length by itself is 5. If we left var i = str.length, when the for loop runs, it will start at index 5, which is undefined if we pass in the string "hello". The last index of "hello" is 4.

Step 3: We want to add each iteration of the for loop to our empty reverseStr string. We achieve that with the following code:

reverseStr += str[i];
