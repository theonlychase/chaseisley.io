Angular 2 Quickstart App
by Chase Isley
Project Description
This app is based off of the tutorial from the official Angular 2 website. The code includes all of my notes, which should help readers along the way. This app covers the following core fundamentals of Angular:

- Built-in directives to show/hide elements and display lists of hero data.
- Create several Components to display hero details and to show an array of heroes.
- One-way data binding for read-only data.
- Editable fields to update a model with two-way data binding.
- Bind component methods to user events like key strokes and clicks.
- Select a Hero from a Master list and edit that hero in the details view.
- Format data with pipes.
- Create a shared service to assemble the heroes.
- Routing to navigate among different views and their components
