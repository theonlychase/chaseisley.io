Angular 2 Quickstart App
by Chase Isley
Project Description
This app is a simple clone of Youtube that implements many features of AngularJS, including:

- Built-in directives
- Custom Directives, including angular-youtube-embed: Angular Youtube Embed
- $http requests to a youtube public API: Youtube Grabber
- One and two way data binding
- Routing with ui-router
- Services
- and more!
