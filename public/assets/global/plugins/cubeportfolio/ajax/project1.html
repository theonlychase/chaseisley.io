Food Truck Finder
by Chase Isley, Blake Hagen, Nick Peterson & Tyler Murdock
Project Description
This app is awesome! The Food Truck Finder was created as a group project at DevMountain web development bootcamp. It was completed in 2 weeks using the Ionic Framework. Authentication was built using Passport Jwt (JSON Web Tokens). We used the Google Maps API for the map and geolocation.

The Food Truck Finder is not available yet on the iOS and Android App Stores, but will be soon. For now, you can view and use the app by downloading the Ionic View App for Android or Apple iOS.

Once downloaded, you will create a free account with Ionic, then you will input our App Code to use the Food Truck Finder.
